# Resume Optimizer

Single-page resume builder with ATS optimization, import/export, and fine-grained typography controls.

## Features

- Import existing resumes (PDF, DOCX)
- Export optimized resumes (PDF)
- ATS-friendly formatting
- Live preview with real-time editing
- Fine-grained typography: font size, line height, paragraph spacing, margins
- Enforces single-page output
- Clean, professional templates
- Section reordering via drag-and-drop

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python run.py
```

Open http://localhost:5000
